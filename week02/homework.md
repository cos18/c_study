5. 4번 문제에서 결과값의 자료형이 결정되는 근거는 무엇인가? (출처 - ch.4 연습문제 Q3)

```C
    int main(void){ 
        double num = 100;
        printf("내 수학 점수는 %d점!", num)
        char a = 'A+';

    }
```

### 출력 결과를 예상해보시오. (빌드하지 말고 머리속으로 생각해보세요.)

```C
    #include <studio.h>
    int main(void)
    {
        int a, b=11, c;
        a = (b++) / 3.0;
        c = (--a) * (b--);
        b /= (float)a;
        printf("%d %d %d", a, a+b, a+b+c);
    }
```
